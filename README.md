# 2021-22-Ionic-Hybrid-App

## Getting started

Install Ionic CLI:
```shell
npm install -g @ionic/cli
```

Install dependencies:
```shell
npm install
```

Serve Ionic as web app:
```shell
ionic serve
```

Ionic docs:
https://ionicframework.com/docs/

Ionic Vue tutorial:
https://ionicframework.com/docs/vue/your-first-app
